# StudyAI

An AI-powered study timetable generator that creates intelligent, difficulty-weighted study schedules. Built with React, TypeScript, and Tailwind CSS.

## Features

- **Adaptive Intelligence** — AI-driven study plans that adapt to your difficulty levels and time constraints.
- **Smart Timetables** — Generate optimized study schedules with built-in breaks and difficulty weighting.
- **Export & Share** — Download timetables as PNG, JPG, or PDF, or share them instantly.
- **Responsive Design** — Works seamlessly on desktop and mobile devices.

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Framer Motion

## Getting Started

```sh
# Clone the repository
git clone https://github.com/ATS-001/studyai-2026.git

# Navigate to the project directory
cd studyai-2026

# Install dependencies
npm install

# Start the development server
npm run dev
```
